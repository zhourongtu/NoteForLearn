**1. Args的最终使用版本**
``` Java
pattern = "l,p#,d*"
Argsarg = new Args(pattern, args)

可以获取所有的值
Argsarg.getBoolean('l')
Argsarg.getInt('p')
Argsarg.getString('d')
```

- 2.针对变化点做行为抽象，解决具体使用抽象时的问题（类型转换、数据存储、处理、异常类型与使用）。
